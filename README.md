# DataBase_SemesterWork
База данных продуктовой сети "Пятерочка" разделена на 3 схемы. Схема core содержит базовые справочники: города, магазины, категории товаров, клиентов и товары. Схема operations включает операционные данные: заказы, доставки, сотрудников, курьеров и остатки товаров. Схема promo предназначена для маркетинговых акций и связей товаров с акциями.

Ключевые особенности базы включают строгие ограничения целостности данных с помощью CHECK. Для телефонных номеров установлена проверка формата +7XXXXXXXXXX, для email - валидация стандартного формата, для имен и фамилий - разрешены только буквенные символы. Все цены и количества имеют проверки на положительные значения. Для временных данных используются типы с таймзоной (TIMESTAMPTZ).

Кроме того для лучшей производительности были установлены индексы для ключевых полей: штрих-коды, даты заказов. Особое внимание уделено составным индексам для таблиц связей многие-ко-многим.

База заполнена тестовыми данными в количестве более 15,000 строк, с акцентом на таблицы-факты: клиенты (1500+), товары (1200+), заказы (2000+), позиции заказов (6000+). Данные генерировались случайно при помощи наборов перестановок разных значений, но при этом значения полей соответствуют минимальной логике реальных данных.
